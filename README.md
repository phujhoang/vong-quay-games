# Vòng Quay Games

Four Vietnamese classroom wheel games, ready for **GitHub Pages**.

## Play

After enabling GitHub Pages (Settings → Pages → Deploy from branch `main` / root):

- Hub: `https://phujhoang.github.io/vong-quay-games/`
- Lớp 1–2: `.../vong-quay-tieng-viet-lop1-2/`
- Giáo lý 3: `.../vong-quay-giao-ly-lop3/`
- Kinh Thánh: `.../vong-quay-kinh-thanh/`
- Việt Nam: `.../vong-quay-viet-nam/`

## Games

| Folder | Description |
|--------|-------------|
| `vong-quay-tieng-viet-lop1-2` | Vietnamese grades 1–2 |
| `vong-quay-giao-ly-lop3` | Catechism grade 3 |
| `vong-quay-kinh-thanh` | Bible phrases |
| `vong-quay-viet-nam` | Vietnamese culture |

## Local

Open `index.html` or serve the folder with any static server.
